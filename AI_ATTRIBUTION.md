# Generative AI Attribution & Usage Declaration

In line with the academic integrity guidelines for DBS and this module (B9IS123), this document outlines how I used Generative AI (Gemini) as an assistant during the development of this web based IT Asset Manager project.

## Where I Used AI Support

* Debugging & Code Fixes: I used AI to help me troubleshoot the dynamic DOM updates for the dashboard metric cards in `backend/static/app.js` and to help resolve some minor layout bugs within the website interface.
* Git & Deployment Issues: AI helped me work through local Git merge conflicts and sync errors when pushing the code to production.
* GCP Deployment Configuration: I used AI to help draft and configure the `Dockerfile` settings so the website could run smoothly as a stateless container on Google Cloud Run.
